## ЭТАП 2: LOW-CODE И NO-CODE {data-track-view-time="nur-course:program:phase2"}

**Цель:** научиться быстро создавать решения с использованием готовых платформ и API.

Здесь ты не просто пишешь код — ты учишься собирать рабочие продукты из существующих инструментов. Это навык, который резко расширяет круг задач, которые ты можешь решить в одиночку или в небольшой команде.

**Модули этапа:**

- **Google API** — Google Sheets, Docs, Forms, Drive. Автоматизация документооборота и сбора данных
- **n8n** — визуальное программирование рабочих процессов. Интеграция сервисов без кода
- **Astro** — современный фреймворк для быстрой сборки сайтов
- **Другие платформы** — по мере формирования программы

*Детальная программа в разработке.*
