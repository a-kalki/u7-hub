## Этап 2: Low-code и No-code {data-track-view-time="nur-course:program:phase2"}

**Цель:** научиться быстро создавать решения с использованием готовых облачных платформ и API — без необходимости поднимать свой сервер.

Здесь ты не просто пишешь код — ты учишься собирать рабочие продукты из существующих инструментов. Всё, что нужно для старта — браузер и интернет.

<span id="module-phase2-automate"></span>

### Автоматизация и интеграции

Связываем сервисы между собой, чтобы они работали без нашего участия.

- **Make.com** (бывш. Integromat) — визуальные интеграции сотен сервисов
- **Zapier** — автоматизация между приложениями
- **n8n Cloud** — визуальное программирование рабочих процессов (облачная версия)

<span id="module-phase2-backend"></span>

### Базы и бэкенд без кода

Храним данные и строим логику без написания серверного кода.

- **Airtable** — базы данных без кода, клиентские интерфейсы
- **Supabase / Firebase** — бэкенд как сервис (BaaS): база данных, авторизация, хостинг
- **Notion API** — базы данных, документация, управление проектами
- **Google Sheets** — таблицы как источник данных для приложений

<span id="module-phase2-app-site"></span>

### Приложения и сайты

Создаём интерфейсы и публикуем результат.

- **Glide** — приложения из Google Sheets за минуты
- **Appsmith / Budibase** — low-code админки и внутренние инструменты
- **Astro** — современный фреймворк для быстрой сборки сайтов, деплой на бесплатные платформы

<span id="module-phase2-comm-pay"></span>

### Коммуникации и платежи

Работаем с пользователями: почта, календари, чат-боты, приём платежей.

- **Google API** — Docs, Forms, Drive, Calendar, Gmail
- **Stripe** — приём платежей (Checkout, Payment Links — без своего сервера)
- **Telegram Bot API** — no-code боты: Manybot, Livegram
