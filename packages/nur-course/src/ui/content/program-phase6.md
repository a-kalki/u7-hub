## ЭТАП 6: КОМАНДНАЯ РАЗРАБОТКА {data-track-view-time="nur-course:program:phase6"}

**Цель:** работа в команде над реальным продуктом для бизнес-заказчика.

Это конечная точка всей программы. Здесь нет учебных задач — есть продукт, который должен работать. Ментор находит реального клиента с реальной бизнес-задачей. Заказчик ставит цель, команда проектирует решение, распределяет задачи, проходит полный цикл: планирование → разработка → ревью → тестирование → деплой.

**Что ты получишь:**
- Опыт командной разработки, близкий к Scrum/Kanban
- Практику декомпозиции бизнес-требований на технические задачи
- Навык презентации и защиты решений перед заказчиком
- Реальный продукт в портфолио, а не учебный проект

Студенты получают опыт, бизнес получает решение бесплатно или за символическую плату.

*Старт первого потока — после набора группы, прошедшей этапы 1–5.*
