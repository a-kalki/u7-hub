## ЭТАП 3: ПРОДВИНУТЫЙ JAVASCRIPT {data-track-view-time="nur-course:program:phase3"}

**Цель:** углубить понимание языка, создать мини-приложение с разделением на UI и API слои.

Здесь ты переходишь от «знаю синтаксис» к «понимаю как язык работает внутри». Итог этапа — работающее браузерное приложение с архитектурным разделением на два слоя. Это твой первый опыт осознанного проектирования, а не просто написания функций.

**Ключевые темы:**
- Замыкание, декораторы, symbols, JSON
- Асинхронность: промисы, `async`/`await`
- Работа с сетью: fetch, REST API на сервис цитат или погоды
- Однопоточость движка js, event-loop
- Серверный JavaScript: Node.js, Bun, npm
- Проектирование двухслойной архитектуры: UI слой и API слой
- Итоговый проект: мини-приложение в браузере

*Детальная программа в разработке.*
