## ЭТАП 5: TYPESCRIPT И ООП {data-track-view-time="nur-course:program:phase5"}

**Цель:** освоить статическую типизацию и объектно-ориентированное программирование. Построить личное приложение.

TypeScript — это стандарт индустрии. Он ловит ошибки до запуска и делает архитектуру явной. ООП даёт инструменты для управления сложностью. Итог этапа — твоё личное приложение (например, игра «Жизнь» или «Сапёр»), возможно с бэкендом.

**Ключевые темы:**
- Система типов TypeScript: примитивы, union, intersection, generics
- Интерфейсы и type aliases
- Основы ООП: классы, наследование, инкапсуляция, полиморфизм
- Паттерны проектирования на практике
- Поэтапное строительство личного приложения

*Детальная программа в разработке.*
