## ЭТАП 1: ОСНОВЫ JAVASCRIPT {data-track-view-time="nur-course:program:phase1"}

**Цель:** изучить JavaScript от синтаксиса до взаимодействия с браузером.

Это фундамент всего курса. Здесь ты пишешь свой первый код, учишься алгоритмически мыслить, осваиваешь Git и GitHub, и создаёшь первые интерактивные страницы. Без этого этапа дальше идти некуда.

### Модуль 1: Синтаксис {#module-phase1-syntax}

Освоить базовый синтаксис JavaScript и написать первые программы. Видео-уроки + самостоятельные задания + 4 практических проекта.

**Проект 1 — Расчёт зарплаты: ИПН и пенсионные отчисления**

Программа принимает имя сотрудника и зарплату, рассчитывает налоги и удержания, выводит результат.

- **Введение в программирование** — код, программа, интерпретатор, поток выполнения
- **Переменные** — let и const, присваивание, область видимости
- **Математические операторы** — +, -, *, /, %, **, сокращённый синтаксис
- **Типы данных** — number, string, boolean, object, null, undefined, typeof, интерполяция
- **Типы данных: исключения и особые случаи** — конкатенация строк, арифметика со строками, TypeError
- **Операторы равенства и сравнения** — ===, ==, <, >, <=, >=, отрицание
- **Логические операторы** — ||, &&, таблицы истинности, составные условия
- **Ветвление** — if / else if / else, цепочки условий
- **Тернарный оператор** — условие ? да : нет
- **Неявное преобразование типов** — автоматическое приведение, falsy-значения
- **Явное приведение типов** — String, Number, Boolean, parseInt
- **Практика: расчёт зарплаты** — применяем всё изученное в одной программе

**Проект 2 — Расчёт зарплаты: групповая обработка**

Дорабатываем программу: добавляем циклы, массивы, объекты. Обрабатываем список сотрудников с подсчётом итогов.

- **Секреты логических операторов** — || и && с небулевыми значениями, короткое замыкание, ??
- **Приоритеты операторов** — таблица приоритетов, ассоциативность
- **Циклы: концепция и основы for** — синтаксис, поток выполнения
- **Циклы: for для перебора строк** — строка как коллекция символов
- **Циклы: while и do-while** — циклы с предусловием и постусловием
- **Массивы (Array)** — создание, методы, перебор
- **Объекты (Object)** — создание, доступ, перебор свойств
- **Map и Set** — ключ-значение и уникальные значения
- **Операторы in, for…in, for…of** — проверка свойств, перебор ключей и значений
- **Практика: групповая обработка зарплаты** — массив сотрудников, циклы, подсчёт итогов

**Проект 3 — Функции для расчёта зарплаты**

Переписываем программу через функции. Учимся декомпозиции: разбиваем большую задачу на маленькие.

- **Функция: синтаксис и return** — объявление, параметры, возврат значения
- **Функция: предикаты и методы реализации** — досрочный return, предикаты
- **Функция: чёрный ящик и декомпозиция** — контракт функции, разбивка на подзадачи
- **Функции: типы и стили объявления** — Declaration, Expression, Arrow, хоистинг
- **Практика: расчёт зарплаты через функции** — calcEmployee и calcDepartment
- **Форматирование таблицы** — padRight/padLeft, динамическая ширина столбцов

**Проект 4 — TDD: пирамида, ёлка и reverseText**

Учимся писать тесты до реализации (TDD). Создаём функции, которые рисуют фигуры в консоли и переворачивают строки.

- **Область видимости и время жизни** — глобальная, функциональная, блочная область
- **Функции: аргументы и передача данных** — по значению vs по ссылке
- **Функции: продвинутые возможности** — вложенные функции, замыкания, колбэки
- **Callback-функции** — синхронные и асинхронные колбэки
- **Рекурсия** — база и шаг рекурсии, стек вызовов
- **Модули: import и export** — разделение кода на модули
- **Мастер-класс: рисуем пирамиду** — ментор показывает buildPyramid(n) вживую
- **Практика: рисуем ёлку в консоли** — drawTree(height), вложенные циклы, декомпозиция
- **Практика: reverseText через TDD** — тесты до реализации, итеративно и рекурсивно

### Модуль 2: Алгоритмика {#module-phase1-algorithm}

Прокачать алгоритмическое мышление через реализацию ~50 функций-аналогов string/array API. Изучить Git и GitHub на практике. Каждый проект — это ветка, коммиты, Pull Request и код-ревью от сокурсников. Правило: 2 аппрува на PR.

**Проект 1 — Git, TDD и посимвольное сравнение строк**

Освоить Git локально, изучить TDD через bun:test, реализовать функции сравнения.

- **Что такое Git** — установка, конфигурация, ключевые понятия
- **Первый репозиторий** — git init, .gitignore
- **TDD и функция len()** — цикл red→green→refactor, bun:test
- **Обработка ошибок** — throw, try-catch, проверка типов
- **Таблица символов** — ASCII, Unicode, лексикографический порядок
- **isEqual()** — посимвольное сравнение строк
- **isNotEqual, isMore, isLess** — лексикографическое сравнение по кодам
- **isMoreOrEqual, isLessOrEqual** — комбинации операторов
- **Финальный прогон** — bun test, git log, commit messages

**Проект 2 — GitHub и первый Pull Request**

Освоить GitHub: аккаунт, SSH, push, PR, ревью. Пушится код из проекта 1.

- **Регистрация на GitHub и SSH-ключ** — аккаунт, ssh-keygen
- **Удалённый репозиторий и git push** — remote add, push
- **Pull Request: создание и описание** — заголовок, changelog
- **Приглашение ревьюеров** — конструктивные комментарии, правило 2 аппрувов
- **Доработка по ревью** — ответы на комментарии, правки

**Проект 3 — Ветки Git и поиск подстрок**

Освоить ветки Git. Реализовать алгоритмы поиска подстрок.

- **Ветки в Git** — branch, switch, параллельная разработка
- **Слияние и конфликты** — merge, fast-forward, разрешение конфликтов
- **indexOf()** — наивный поиск подстроки: вложенные циклы
- **includes(), startsWith(), endsWith()** — проверки наличия
- **reverse()** — переворот строки
- **repeat()** — повторение строки
- **Пуш ветки и PR** — git push -u origin
- **Ветка для исправлений** — fix-ветка, resolved-комментарии
- **Финальный мерж** — после 2 аппрувов

**Проект 4 — JSDoc и трансформация строк**

Освоить JSDoc. Реализовать 8 функций трансформации: `substring`, `slice`, `trim`, `replace`, `replaceAll`, `pad`, `upperCase`, `lowerCase`.

**Проект 5 — Массивы: структура и базовые операции**

Понять мутабельность. Реализовать 9 функций: `len(arr)`, `at`, `push`, `pop`, `unshift`, `shift`, `indexOf`, `lastIndexOf`, `includes`.

**Проект 6 — Массивы: трансформация**

Реализовать 7 функций: `fill`, `reverse(arr)`, `concat`, `slice(arr)`, `join`, `splice`, `flat`.

**Проект 7 — Итеративные методы: forEach, map, filter**

Callback-паттерн на практике. 5 методов: `forEach`, `map`, `filter`, `some`, `every`.

**Проект 8 — Итеративные методы: find, reduce, flat**

Продвинутые методы. 4 функции: `find`, `findIndex`, `reduce`, `flat`.

**Проект 9 — Бизнес-утилиты**

Применить всё в реальных задачах. `getQueryParams`, `parseUrl`, `toCSV`, `fromCSV`, `groupBy`.
