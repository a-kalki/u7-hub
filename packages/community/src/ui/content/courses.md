## ОБРАЗОВАНИЕ ПО-НОВОМУ {.sw-bg-color data-track-view-time="community:courses:new-education"}
# 

Мир изменился. Старые методы подготовки специалистов устарели, и мы ищем новые подходы. Наши курсы — альтернатива традиционному образованию, где главной ценностью является опыт ментора, прошедшего путь от новичка до профессионала.

**Что мы меняем:**

| По-старому | По-новому |
|---|---|
| 3–4 года ради диплома | 12–18 месяцев ради реальных навыков |
| Программа написана министерством | Программа написана практикующими разработчиками |
| Преподаватель без опыта в индустрии | Ментор, прошедший путь от новичка до профессионала |
| Учебные задачи из учебника | Реальные задачи от реальных заказчиков |
| Экзамен проверяет память | Код-ревью проверяет умение |
| Обратная связь — раз в семестр на зачёте | Обратная связь — каждый проект через ревью |
| Диплом говорит «отучился», но не говорит насколько ты хорош | Рекомендация ментора и сокурсников описывает твои реальные качества |
| Ты в одиночку, твоя цель — оценка | Ты в команде, ваша цель — работающий продукт |
| Сдал экзамен и забыл предмет, потому что не применяешь | Сдал проект и поддерживаешь его работу, потому что продукт живёт |
| Курсовая, которую никто не читает и не запускает | Каждый проект — обязательная ступень к следующему. Без неё не взять более сложные концепции |
| Ты не влияешь на то, кто с тобой в группе | Студенты влияют на допуск к следующим модулям |
| Бизнес-задачи — это «когда-нибудь потом» | Студенты решают бизнес-задачи, бизнес получает решение бесплатно |

**Какие навыки ты наработаешь:**

- **Освоишь синтаксис и продвинутые концепции языка** — будешь уверенно писать работающий код
- **Решишь много алгоритмических задач** — научишься переводить задачу с человеческого языка в работающую программу
- **Освоишь Git и GitHub** — научишься не терять свой код при изменениях, делиться им и работать совместно в команде
- **Освоишь парадигму работы в TDD** — будешь более уверенным что твой код работает как нужно тебе
- **Будешь учиться проектированию архитектуры приложений** — строить сложные приложения которые работают и готовы развиваться
- **Пройдёшь десятки код-ревью** — научишься читать чужой код, аргументировать решения и спокойно принимать критику
- **Поработаешь в команде над реальным продуктом** — узнаешь как договариваться о решениях и отвечать за общий результат

> Старая школа растит теоретиков и исполнителей на учебных материалах. Мы растим профессионалов — тех, кто умеет думать и делать, — на реальной работе.

Нужна определённая смелость, чтобы не пойти по проторенной дороге за дипломом, а выбрать путь, где важны не бумаги, а навыки. Мы предлагаем такую альтернативу. Но выбирать тебе.
