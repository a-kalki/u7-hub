# Specification: Documentation Update (IT Community Integration)

## 1. Overview
Проект расширился: помимо курсов программирования, важной частью стало развитие IT-сообщества. Текущая документация в `conductor/` не полностью отражает эту часть проекта. Задача трека — актуализировать документацию, добавив информацию о сообществе и его образовательных модулях.

## 2. Functional Requirements
- **Update `conductor/product.md`**: 
    - Добавить миссию развития IT-сообщества в регионе.
    - Отразить цели, связанные с подготовкой специалистов и поддержкой бизнеса через сообщество.
- **Update `conductor/tech-stack.md`**:
    - Описать структуру папки `src/community`.
    - Уточнить архитектуру проекта (наличие статических модулей сообщества).
- **Create `conductor/community.md`**:
    - Создать новый документ, описывающий концепцию IT-сообщества.
    - Включить информацию о 4 модулях обучения: Программист, Фриланс, Разработчик, Командник.
    - Описаны разделы (Миссия, Программа, Сообщество, Спонсорам и т.д.).

## 3. Acceptance Criteria
- Документация в `conductor/` полностью охватывает все модули проекта (`course` и `community`).
- Описаны 4 образовательных модуля сообщества.
- Структура файлов в `src/community` отражена в техническом стеке.
- Ссылки на новые документы добавлены в `conductor/index.md`.

## 4. Out of Scope
- Рефакторинг кода в `src/community`.
- Изменение дизайна лендингов.
- Детальное описание учебных планов (только обзорно).
